# zeppelin-csv-spell 

Parse CSV to table based on [papaparse](http://papaparse.com/)

## Usage

![](https://raw.githubusercontent.com/1ambda/zeppelin-csv-spell/master/screenshots/usage.gif)

## Example Paragraph for Testing

```
%csv

date,bucket,count
2017-02-20,800,119
2017-02-21,900,123
2017-02-22,1000,173
2017-02-23,1100,226
2017-02-24,1200,284
2017-02-25,1300,257
2017-02-26,1400,268
2017-02-27,1500,244
2017-02-27,1600,220
```
